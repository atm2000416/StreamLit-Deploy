---
name: search-debug
description: Debug camp search issues from trace logs. Use when the user
  pastes trace logs, reports wrong results, or asks why a search failed.
---
When analyzing a trace log or search issue:

1. Diagnose the current issue (which path fired, what went wrong)
2. Check if this matches any known fix in FIXES.md
3. If it's a NEW issue, flag it: "This looks like a new bug — not covered by existing fixes"
4. Suggest what Fix #N would look like
5. Check for related edge cases that might have the same root cause
